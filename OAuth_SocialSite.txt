1. Right Click project--> Press F4--> Update SSL Enable to True
2. Copy the Https URL
3. Right Click Project-->Property-->Web-->Paste Https URL in Project URL
4. developers.facebook.com-->register and login-->will get Add a new App

6. Create New facebook App ID
